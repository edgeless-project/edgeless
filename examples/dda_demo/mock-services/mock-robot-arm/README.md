# Simulated Actor Robot Arm

Models a mock actuator that moves a robot arm; uses dda as library.
It makes use of the dda as library.

## Prerequisite

A current version of `go` installed.

## How to build

## How to run

MQTT configuration hard-coded in main.go.

Run with:

```bash
go run main.go
```
